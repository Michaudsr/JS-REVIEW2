## HTML

HTML  is the standard markup language for documents designed to be displayed in a web browser

## HTML BoilerPlate
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
