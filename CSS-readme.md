## CSS 
Css is a stylesheet language used for describing the presentation of a document written in a markup language like HTML

## CSS Example
```
h1 {
    color: blue;
    font-family: Times;
    margin: 0;
    text-align: center;

}

p {
    color: black;
    font-family: Times;
    text-align: center;
    margin: 30px;
}
```
